## 功能特性

### 日历排课

- **月历视图**：完整的月历展示，在日历上直接查看每日课程安排
- **添加课程**：点击任意日期即可选择添加新课程，智能检测时间冲突
- **课程处理**：点击已有课程或者日期可选择编辑复制粘贴删除操作，智能检测时间冲突或重复
- **月份导航**：支持左右箭头快速切换月份，或使用下拉菜单快速跳转到指定年月
- **节日调休**：集成中国农历节日和相关调休信息
- **支持多选**：日期格子支持鼠标左键按住拖拽框选或者按住ctrl键+鼠标左键点击进行多选，课程标签只支持按住ctrl键+鼠标左键点击进行多选

### 学生管理

- **学生管理**：查看、添加、编辑、删除学生信息
- **学生搜索**：支持通过姓名快速搜索学生
- **机构管理**：自定义机构名称，支持增删改，可为每个机构分配颜色标签
- **年级管理**：自定义年级名称，支持增删改，可为每个年级分配颜色标签
- **预设费用**：可预设一对一课型的课时费，排课时自动加载计算

### 费用统计

- **年月筛选**：按年份和月份筛选统计数据
- **机构筛选**：按机构筛选统计数据
- **统计卡片**：各种数据信息一目了然
- **数据导出**：将数据导出为 HTML 文件，方便存档和分享


### 用户体验

- **主题切换**：支持浅色/深色/自动三种主题模式
- **流畅动画**：页面切换、模态框、下拉菜单、主题切换等交互有流畅动画效果
- **操作反馈**：友好的通知提示，服务器连通指示器
- **开源字体**：使用霞鹜文楷（LXGW WenKai）开源中文字体

## 使用指南

### 登录与注册

1. 打开系统
2. 使用已注册账号登录、也可使用试用账号：`test`（密码为空）
3. 登录后数据自动从云端同步

### 添加基础数据

1. 进入"学生管理"页面
2. 先通过"机构管理"添加机构（如"本校"、"分校"等）
3. 再通过"年级管理"添加年级（如"一年级"、"初二"等）

### 添加学生

1. 进入"学生管理"页面
2. 点击"添加学生"按钮
3. 填写姓名、选择机构和年级
4. 设置该学生的一对一课时预设费
5. 点击保存

### 安排课程

1. 在日历视图中点击想要安排课程的日期
2. 点击"添加课程"按钮
3. 选择课型（一对一或多人课）
4. 选择上课学生
5. 设置开始时间和时长
6. 多人课还需自定义费用
7. 可添加备注信息
8. 点击保存

### 查看统计数据

1. 进入"费用统计"页面
2. 选择要查看的年份和月份
3. 如需按机构筛选，选择对应机构
4. 查看各项统计数据
5. 点击"导出数据"可导出为 HTML 文件

## 注意事项

1. **网络要求**：登录、数据同步等功能需要网络连接
2. **试用模式**：试用模式下数据不会被保存，刷新或关闭后数据会丢失
3. **浏览器支持**：建议使用现代浏览器（如 Chrome、Firefox、Edge、Safari 等）获得最佳体验
4. **数据安全**：请妥善保管账号密码，不要在公共电脑上保存登录状态
5. **意外防范**：尽量不要在多端登录后同时进行数据编辑，以防发生意料之外的数据覆盖或丢失
